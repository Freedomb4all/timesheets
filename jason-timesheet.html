Pay periods run Sun–Sat. Each period is calculated automatically from your time records and synced to the Payroll Periods tab in your Google Sheet at the end of each pay period.
1

2
Go to Extensions → Apps Script, paste the code below, then click Deploy → New deployment → Web app. Set "Execute as" to Me and access to Anyone.
3
Copy the Web App URL from the deployment and paste it below.
